RootKit Hunter

rkhunter (Rootkit Hunter) is a Linux security tool used to detect signs of rootkits, backdoors, and local exploits.

  • Scans for rootkits
    Looks for known rootkit signatures (files, directories, strings).
  • Checks system binaries
    Compares important binaries (ls, ps, netstat, etc.) against known-good hashes to detect tampering.
  • Detects hidden files
    Rootkits often hide files/processes ‚Äî rkhunter looks for anomalies.
  • Checks permissions and configs
    Flags suspicious file permissions (e.g., writable by everyone, SUID root).
  • Warns about vulnerabilities
    Looks for outdated software versions that could be exploited.
